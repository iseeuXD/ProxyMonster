### ⚡ **ProxyMonster**, çeşitli proxyleri kontrol etmek için geliştirilmiş bir Go programıdır.

#### 🌟 Desteklediği proxy türleri:
- **HTTP**
- **HTTPS**
- **SOCKS4**
- **SOCKS5**

---

## 🚀 Özellikler
✅ Eşzamanlı (Concurrent) proxy kontrolü    
✅ Terminal arayüzü.  
✅ En hızlı proxy checker!

---

## 📦 Gereksinimler
- **Go**: `1.18` veya daha üst sürüm.

---
